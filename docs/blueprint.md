# **App Name**: StarPals Casting AI

## Core Features:

- Actor Profile Creation: Create and store actor profiles with details like name, age, physical attributes, skills, emotional traits, portfolio URL, and image URL.
- Role Definition: Define roles with project name, character name, age range, physical requirements, required skills, and emotional traits.
- Compatibility Scoring Tool: Calculate a compatibility score (0-100) between actors and roles based on age match (30%), physical attributes (40%), and skills & traits (30%) using a generative AI powered tool.
- Ranked Actor Matches: Retrieve a list of actors ranked by compatibility score for a given role.
- Data Seeding: Seed the database with 5 actors and 3 roles to showcase matching capabilities and facilitate initial testing and demonstration of the StarPals app.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) which translates to a vibrant blue (#3399FF), inspired by the entertainment industry.
- Background color: A desaturated, lighter shade of the primary hue at HSL(210, 20%, 90%), or #E5F0FF.
- Accent color: HSL(180, 60%, 40%) for visual interest: a turquoise (#339980).
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clear, card-based layouts for actor and role profiles to enhance readability and visual appeal.
- Incorporate icons to represent skills and traits, enhancing user experience and interface intuitiveness.
